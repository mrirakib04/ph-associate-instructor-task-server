# Server - BookWorm - Associate Instructor Task

The BookWorm backend is built using **Node.js**, **Express.js**, and **MongoDB**.
It provides RESTful APIs for fetching the latest books, retrieving individual book details, and managing book data.
The server is optimized for modern frontend frameworks like **Next.js**.
